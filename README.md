# machinegun_ios


Install `https://apps.apple.com/ru/app/testflight/id899247664`
Join to `https://testflight.apple.com/join/97i7KM8O`

Execute the command:
* ` apk update`
* ` apk upgrade`
* ` apk add git`
* ` git clone https://github.com/realo/machinegun_ios`
* ` cd /machinegun_ios`
* ` sh install.sh`
* ` python machinegun_ios.py`
